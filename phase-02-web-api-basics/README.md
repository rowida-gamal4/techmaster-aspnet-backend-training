## Phase Overview
This phase focuses on learning how to build RESTful APIs using ASP.NET Core Web API.

Topics covered include:

- Web API fundamentals
- Controllers
- Routing
- HTTP methods
- Model Binding
- Validation
- Dependency Injection
- CRUD operations
- Swagger / OpenAPI
- Postman
- API design best practices
