# Task 00 - API Workspace Setup

## Objective

The goal of this task is to prepare a clean ASP.NET Core Web API workspace that will be used throughout Phase 02. The project is configured with controllers, Swagger/OpenAPI support, and HTTPS redirection.

## Technologies

- .NET 8
- ASP.NET Core Web API
- Swagger / OpenAPI

## Features

- ASP.NET Core Web API project created
- Controllers enabled
- Swagger enabled
- HTTPS redirection enabled
- Ready for future API development

## How to Run

### Using the .NET CLI

1. Open a terminal inside the project directory.
2. Run the application: dotnet run
3. Open Swagger in the browser or use Postman
ex : http://localhost:5192/swagger/index.html
(change the port )
