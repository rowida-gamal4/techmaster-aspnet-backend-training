# Phase 01 - Backend Foundations

This repository contains all the tasks I completed during Phase 01 of the TechMaster ASP.NET Backend Career Training.

The purpose of this phase was to build a strong foundation in C#, Object-Oriented Programming (OOP), LINQ, SQL, Git, and problem-solving before moving to ASP.NET Core.


# Technologies Used

- C#
- .NET
- LINQ
- SQL Server
- Git & GitHub
- Visual Studio Code



# Tasks

## Task 01 - C# Basics

Practiced the fundamentals of C# including variables, operators, conditions, loops, methods, and solving basic programming problems.



## Task 02 - Bank Account System

Built a simple banking system using Object-Oriented Programming concepts such as classes, objects, encapsulation, constructors, and methods.



## Task 03 - Product Catalog with LINQ

Created a product catalog application and used LINQ to perform filtering, sorting, grouping, searching, and other data operations.



## Task 04 - Collections Practice

Practiced working with C# collections such as Lists and Dictionaries while improving problem-solving skills.



## Task 05 - Debug & Refactor Pack

Refactored an existing console application by separating responsibilities into different classes, improving code readability, adding validation, removing magic numbers, and organizing the business logic without changing the original functionality.



## Task 06 - SQL & ERD Starter

Designed relational databases from business requirements by creating ERDs, defining primary and foreign keys, building SQL tables, inserting sample data, and writing SQL queries using joins, grouping, and aggregate functions.

Scenarios included:

- Library Management System
- Simple Store & Orders System
- Training Center Registration System



## Task 07 - Interview Answers & Explanation Pack

Prepared short interview-style answers covering the concepts learned during Phase 01, including OOP, LINQ, SQL, Git, and software design principles.



# What I Learned

During this phase I learned how to:

- Write clean and organized C# code.
- Apply Object-Oriented Programming principles.
- Separate business logic from the user interface.
- Use LINQ to query collections.
- Design relational databases and ERDs.
- Write SQL queries with JOIN, GROUP BY, and aggregate functions.
- Use Git and GitHub to manage projects.
- Document projects using README files.
- Refactor code without changing its functionality.
